FlowCSS Documentation

FlowCSS is an animation-first CSS utility framework with zero build requirements. Add one CSS file, optionally one JS file, and unlock 200+ utility classes including 40+ animations, scroll reveal, glassmorphism, magnetic buttons, and full component primitives.

Installation

via npm

npm install flowcss
Then import in your project: 
/* In your main CSS or JS file */
import 'flowcss/dist/flow.css';
import 'flowcss/dist/flow.js'; // optional — for JS features

Direct HTML

<link rel="stylesheet" href="flow.css">
<script src="flow.js" defer></script>

CDN Usage

Use FlowCSS directly from jsDelivr without installing anything:

<!-- CSS -->
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/flowcss@1/dist/flow.min.css">

<!-- JS (optional) -->
<script src="https://cdn.jsdelivr.net/npm/flowcss@1/dist/flow.min.js"
        defer></script>

Publishing FlowCSS on NPM

Follow these steps to publish your own fork or a derived framework to the npm registry.

  1. Create an npm account
    Go to npmjs.com/signup and register. Verify your email.
  2. Login locally
    npm login
# Enter your username, password, and OTP if 2FA is on
  3. Set up package.json
    {
  "name": "flowcss",
  "version": "1.0.0",
  "description": "Animation-first CSS utility framework",
  "main": "dist/flow.js",
  "style": "dist/flow.css",
  "files": ["dist", "README.md"],
  "keywords": ["css", "animation", "utility", "framework"],
  "license": "MIT",
  "repository": {
    "type": "git",
    "url": "https://github.com/yourname/flowcss"
  },
  "scripts": {
    "build": "node scripts/build.js",
    "prepublishOnly": "npm run build"
  }
}
  4. Build your dist files
    Run the build script to minify CSS and JS into dist/: 
    npm run build
  5. Publish
    npm publish --access public
    For scoped packages (e.g., @yourname/flowcss), make sure the name in package.json starts with your npm scope.
  6. Updating versions
    npm version patch   # 1.0.0 → 1.0.1 (bug fix)
npm version minor   # 1.0.0 → 1.1.0 (new feature)
npm version major   # 1.0.0 → 2.0.0 (breaking change)
npm publish
Tip: Add a .npmignore file to exclude src/, docs/, and node_modules/ from the published package. Only dist/ needs to be published.

Layout & Flex

FlowCSS includes powerful shorthand layout classes for flex and grid that eliminate repetitive declarations.

ClassCSS Output
f-centerdisplay:flex; align-items:center; justify-content:center
f-betweendisplay:flex; align-items:center; justify-content:space-between
f-coldisplay:flex; flex-direction:column
f-grid-3display:grid; grid-template-columns:repeat(3,1fr)
f-grid-autoauto-fill grid with min 200px columns
f-staggerScroll-triggered staggered children (JS)

Example

<div class="f-grid-auto f-gap-6">
  <div class="f-card">Card 1</div>
  <div class="f-card">Card 2</div>
  <div class="f-card">Card 3</div>
</div>

Scroll Reveal

Elements animate into view as you scroll. Powered by Intersection Observer in flow.js.

ClassEffect
f-revealFade + slide up
f-reveal-leftFade + slide from left
f-reveal-rightFade + slide from right
f-reveal-zoomFade + scale up
f-reveal-flipFade + Y-axis flip
f-repeatRe-animates every time element enters viewport
 
<div class="f-reveal" data-flow-delay="200">
  Animates on scroll with 200ms delay
</div>

<div class="f-reveal-zoom f-repeat">
  Animates every time it enters viewport
</div>

Stagger Children

Apply f-stagger to a parent element. When it enters the viewport, each child animates in with an increasing delay automatically.

<div class="f-stagger f-grid-3 f-gap-4">
  <div class="f-card">Appears first</div>
  <div class="f-card">Appears second</div>
  <div class="f-card">Appears third</div>
</div>
Stagger supports up to 10 children with 80ms between each. Children 11+ share the last delay.

Magnetic Buttons

Adds a physics-based cursor attraction to buttons. The element follows the cursor within its bounds on hover.

<button class="f-btn f-btn-primary f-magnetic">
  Magnetic Button
</button>

3D Card Tilt

Real-time mouse-tracked 3D tilt perspective on any element. Add an inner element with f-tilt-inner for a floating depth layer.

<div class="f-card f-tilt">
  <h3 class="f-tilt-inner">Floating layer</h3>
  <p>Move your cursor over this card</p>
</div>

Glassmorphism

Three glass variants with backdrop blur — works best on colorful backgrounds.

<div class="f-glass f-p-6 f-rounded-xl">Glass card</div>
<div class="f-glass-dark f-p-6 f-rounded-xl">Dark glass</div>
<div class="f-glass-primary f-p-6 f-rounded-xl">Primary tinted</div>

Count-Up Numbers

Animate numbers from 0 to a target value when they scroll into view.

<span
  data-flow-count="5000"
  data-flow-prefix="$"
  data-flow-suffix="+"
  data-flow-duration="2000"
  data-flow-decimals="0"
>0</span>

Dark Mode

Toggle dark mode by adding f-dark-mode to <html> or <body>. Use the JS helper or the built-in toggle function.

<!-- Toggle button -->
<button onclick="flowToggleDark(this)">🌙 Dark Mode</button>

/* Or in JavaScript */
window.FlowCSS.toggleDark();

State is persisted in localStorage automatically.

VSCode Extension

The FlowCSS VSCode extension is included in the project under vscode-extension/. It provides:

  • Autocomplete for all f-* classes
  • Hover documentation showing the CSS output
  • Snippet library for cards, buttons, grids, and animations
  • Color previews for gradient utilities

Installing the extension locally

  1. Open the vscode-extension/ folder in VSCode
  2. Run npm install to install the vsce toolchain
  3. Press F5 to launch the Extension Development Host
  4. To package: npx vsce package — produces a .vsix file
  5. Install the .vsix: Extensions panel → ⋮ → Install from VSIX

Publishing to VS Marketplace

  1. Create a publisher at marketplace.visualstudio.com/manage
  2. Generate a Personal Access Token (PAT) with Marketplace scope
  3. Login: npx vsce login yourpublisher
  4. Publish: npx vsce publish
